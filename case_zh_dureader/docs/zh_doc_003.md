# 中文复杂检索文档

## 背景材料1
一、解疑惑,板栗可以生吃吗 其实板栗是可以生吃的,并不是像大家所说的什么对身体有影响或者是有细菌,相反生吃栗子还是对身体有一些益处的。 首先板栗它有很多营养,长年吃时可以缓解腰酸背痛等症状,如果是老年人的话,一天吃三到五颗生栗子就可以了,可以很好的缓解身上的这些病痛。 其次生板栗当中含有非常多的维生素,氨基酸,维生素,和各种微量元素,所以我们在吃的时候呢,完全可以补充身体的营养,达到营养的均衡作用的。 再次生板栗含有非常多的淀粉,可以吃饱肚子,而且同时它所含的油脂类会比较少,碳水化合物却非常高,因此对减肥人士来说,吃生栗子是有减肥的功效的,在不饿肚子的情况下就可以轻轻松松的减肥了。 二、板栗还可以怎样吃 那么除了直接生吃的话,其实更多时候还是希望可以把它煮熟了吃,我们可以把栗子做成糕点,菜肴煮粥等等,最受欢迎的当然都是一些糖炒栗子,或者是栗子烧鸡,栗子烧肉等等,或者是我们自己用冰糖烧栗子等等这种好吃的烧法也是可以的。所以除了生吃,各种煮熟的吃法也可以让身体得到充分的营养补充,同时满足口欲。 因此生吃板栗是没有任何问题的,小伙伴们需要再担心生栗子会对身体有影响了,当然了我们在吃栗子的时候,也要适可而止,适量的去吃,不要吃太多太多,不然的话容易肚子被撑着。今天的回答希望对于各位小伙伴有所帮助。

## 背景材料2
一、新鲜板栗可以生吃吗 新鲜板栗是可以生吃的,生食具有更好的补肾效果。医药学家孙思邈就说板栗是“肾之果”,指出“生食之”可治疗腰脚不遂。但儿童、脾胃虚寒、产妇、便秘、肠胃功能不佳者不适合生吃板栗,在日常生活中也不适合食用过多,极容易造成腹胀或者是腹泻等消化不良的现象。 生吃栗子味道比熟栗子更甜,口感比较清脆,不像熟栗子这么软糯。适量的生吃栗子可以有效的健脾和胃,并且促进新陈代谢正常进行,能够吸收生栗子中的糖分,从而增强人体的免疫力,一般正常人每天食用5-10粒即可。 二、板栗的保管贮藏 板栗虽属坚果,但是在保管过程中却怕冻、怕热、怕干和怕水。如保管不当极易引起失水、发芽和腐烂变质。一般在保管过程中,当麻袋内温度超过20℃时,应立即采取倒垛、串袋通风等降温措施。 我国板栗主要出口日本,并且有着悠久的经营历史每年对日本出口量均在2万吨以上,是我国传统大宗出口商品之一。近年来除主销日本外,还销往韩国、菲律宾、新加坡等国家。 关于板栗我们就说到这里,虽然板栗营养丰富又美味,但是吃起来还是要注意适量,以免过犹不及,对身体产生危害。

## 关键材料
蒸板栗的做法原料：新鲜板栗500克。做法：1、将板栗洗干净，沥干水分。2、用刀在板栗背上切一刀。3、放入蒸锅笼屉上，大火上汽后，转中火15分钟，看到板栗开口即可。补充板栗属于坚果类，但它不像核桃、榛子、杏仁等坚果那样富含油脂，它的淀粉很高。干板栗的碳水化合物达到77%，与粮谷类的 75%相当;鲜板栗也有40%之多，是马铃薯的2.4倍。鲜板栗的蛋白质含量为4%至5%，虽不如花生、核桃多，但略高于煮熟后的米饭。板栗的碳水化合物有40%由淀粉组成，实际上，板栗的淀粉含量是马铃薯的2倍。新鲜板栗富含维生素C和钾，也含有叶酸、铜、维生素B6、镁和维生素B1；每100克新鲜板栗中，水分占52%，另含蛋白质3克、碳水化合物44.2克。煮好的板栗富含钾，也含有维生素C、铜、镁、叶酸、维生素B6、维生素B1、铁和磷；每100克煮熟的板栗中，水分占68.2%，另含蛋白质2克、碳水化合物28克。但某些方面板栗的营养更高于粮食。栗子的维生素B1、B2含量丰富，维生素B2的含量至少是大米的4倍，每100克还含有24毫克维生素C，这是粮食所不能比拟的。人们恐怕很难想到，鲜板栗所含的维生素C比公认含维生素C丰富的西红柿还要多，更是苹果的十多倍!栗子所含的矿物质也很全面，有钾、镁、铁、锌、锰等，虽然达不到榛子、瓜子那么高的含量，但仍然比苹果、梨等普通水果高得多，尤其是含钾突出，比号称富含钾的苹果还高4倍。栗子的营养保健价值虽然很高，但也需要食用得法栗子不能一次大量吃，吃多了容易胀肚，每天只需吃6至7粒，坚持下去就能达到很好的滋补效果。
