# 中文复杂检索文档

## 背景材料1
一、晾衣架有哪些类型 1、落地式:适合租房单身族 落地式晾衣架拆装非常的方便,占地面积比较小,翼式的还可以自由折叠,不用的时候放在门后的空间就好。还有一个最大的好处,搬家时随时都能搬走,只要它还能够使用。雨雪天气和返潮季节都可以搬进室内,关闭门窗,衣服比晾在室外都要干燥很多。 2、壁挂式:适合北方或阳光房 顾名思义,壁挂式晾衣架是装在墙壁上的。平时放下时贴着墙壁,不会占用空间,要晾晒衣服时才撑起来。另外一种是拉伸型,也是装在墙壁上,平时可推紧贴在墙壁上,使用时拉开来晾挂衣服。壁挂式晾衣架可以将衣服完全置于阳光之中,不过,折叠型的悬挂太矮,不美观。 3、升降式:城市蜗居 晾杆是升降晾衣架的主要部件,厚度更是承重的保证。市场上的产品用料也最杂,有塑料、锌合金、铁皮及铝钛合金。最好的当然是优质铝钛合金,加上先进的表面工艺技术处理,基本上是不生锈、不氧化、永不爆裂,能持久亮丽如新。 二、如何正确选择晾衣架 1、手摇器 自动晾衣架的核心部件就是手摇器了,在试摇动时,要注意使用起来是否顺滑、不摇晃、以及有没有难听的噪音。质量可靠的晾衣架则由前后四轴承组成,在升降过程中更加轻便顺畅,无异响,从而增加产品的使用寿命。 2、看粗细 因为晾衣架长期在户外使用,钢丝绳是否防锈、柔软都是关键因素。信誉度高的晾衣架均采用￠1.5mm以上的不锈钢丝绳,三年时间内保证使用柔软不生锈;市场上大部份产品采用的是￠1.2mm不绣钢丝绳或是不锈铁钢丝绳,此类钢丝绳所承受力度不够,使用后表面容易被腐蚀氧化随后生锈。 3、试手感 转向器是升降过程中主要的转动部件,要只要升降是否转动灵活、脱跳槽。好的晾衣架采用高速轴承,在传动过程中减少阻力,使整个升降过程更加轻松;普通产品则采用单一轴承,无钢丝绳防脱跳槽装置,容易造成钢丝绳卡住、缠绕。 4、选材质 自动晾衣架材质大致分为不锈钢、以及合金材质。不锈钢的价格比较便宜,但重量大、易上锈,因此耐用度不高。合金材质又分为铝合金、铝镁合金、钛合金等。它的特点是重量轻、柔韧度较好,而且合金的晾衣架可以在表面做光亮、阳极氧化、粉末喷涂、电泳等处理,使晒衣架更耐用防锈,色泽也更好。
