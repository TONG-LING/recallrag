# 中文复杂检索文档

## 背景材料1
甘油三脂偏低原因很多。常见原因:第①种可能:是饮食造成,一方面可能是平时所吃的食物中含有的脂肪类物质过少,引起的摄入不足;另一方面也可能是自身的消化不良、吃饭过快,导致的脂类物质被机体消化吸收过少。第②种可能:摄入体内的脂肪主要是通过胆汁来分泌和消化吸收的,一旦胆汁分泌异常、胆汁分泌不足时,脂肪便不能被很好地吸收利用,导致血清甘油三酯偏低。第③种可能:一些疾病也可以引起甘油三酯偏低,常见的有肝功能严重障碍、肾上腺皮质功能减退、甲状腺功能亢进等。第④种可能:甘油三酯偏低的原因也可能是在饥饿的时候去做的检查,因为当人体饥饿的时候,体内的甘油三酯会和蛋白质结合,这时候检验便无法检测出甘油三酯。

## 背景材料2
第一,饮食不当可导致血清甘油三酯偏低。 所吃食物中脂肪过少,吃饭速度过快,可出现脂肪吸收过少,导致血清甘油三酯偏低。 第二,胆汁分泌异常导致血清甘油三酯偏低。 当体内胆汁分泌不足时,不能很好的分化食入的脂肪,脂肪不能被很好的吸收,导致体内血清甘油三酯少。 第三,疾病导致血清甘油三酯偏低。 甲状腺功能亢进、肾上腺皮质功能减退及肝功能严重障碍等疾病可出现血清甘油三酯低。 第四,饭前检查可出现血清甘油三酯偏低。 在体检时许多血脂检查多数是饭前抽血,此时血液中的甘油三脂已经与血中的蛋白结合,所测出的检查结果往往要低一些。
