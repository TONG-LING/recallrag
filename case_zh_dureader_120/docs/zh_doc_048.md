# 中文复杂检索文档

## 背景材料1
小口径枪弹面临严峻挑战战争实践表明,75%的步枪交战距离在200米以内,95%在400米以内。美国因此在20 世纪60年代就舍弃了射程远、威力大的7.62×51毫米北约NATO弹药,而转用M16系列步枪来发射中等威力的5.56×45毫米M193弹药。越南战争中,M16A1式步枪武器及5.56毫米弹药比较适合当时的战场环境。M193弹药在近距遭遇战中杀伤力强,能够轻而易举地穿透竹子搭建的房屋墙壁,这也是小口径突击步枪备受称赞的原因和风起云涌发展的开端。然而多年之后在索马里,美军的5.56毫米M855枪弹却不能有效穿透市区的砖墙和其他障碍物。美军特种部队使用M4卡宾枪发射M855时,对敌方有生目标的停止能力非常弱,经常需要多发子弹命中目标才能压制住一个敌人。美军第82空降师和第 101空中突击师的士兵们一直使用M4和M4A1卡宾枪来代替M16A2步枪,在阿富汗和伊拉克战场也遇到了同样的问题。人们不禁要问:发生什么事了,小口径步枪怎么了?

## 背景材料2
提到狙击枪，稍微有点军事常识的军迷都会第一反映就想到“巴雷特”重型狙击枪。被称为狙击之王，全球数十个国家配备该武器，至今在世界范围内没有任何一款狙击枪与其相提并论，不久是在性能参数上碾压同级别对手，还有很多优点让其傲视对手。巴雷特狙击枪是美国以及美国支持的一些联盟国家都在用的反器材武器，军人们习惯的叫它们轻50，多用于特种部队。该枪虽然型号比较多，但是无非是因为性能上的一些改进才使得型号越来越多。M82A1，M82A3，M107，M82A2，M500这些型号的巴雷特狙击枪。但都是些枪托，瞄准具等小的改变，大体上这枪采用的还是12.7毫米的枪弹，不管是不是枪管加长，只不过是增加了精准度，射程不减，都大差不差，最大射程是6800米，有效射程为1850米。所以说基本上不管什么型号，我们普通的军迷叫它巴雷特总没错的，最棒的阻击枪。

## 关键材料
坦克世界中的M8A1自行反坦克炮是在M5坦克基础上专门设计的用于支援步兵的自行火炮。由凯迪拉克公司生产，1944年1月到9月一共生产了1778辆。1942年春天，M3和M5型斯图亚特轻型坦克已经被证实由于只有装备37毫米反坦克炮而导致火力严重不足，因此不适应对德作战。所以美国当时很苦恼，但是美国当时已经装备大量的该型号坦克，因此如何处理这些已经生产的东西将是当务之急。M8A1型坦克歼击车恰好美国陆军对于M2和M3型75毫米榴弹炮非常需要，这种性能良好的榴弹炮重量轻，射速快，精度高，而且便于拆解运输，口径比为18.4倍，全长为3.68米。多完美的设计呀，但是由于美军的机械化进程也较早，所以军方一直希望有一种平台可以搭载这个轻便的火炮，编制在重型坦克营之中，共同近距离支援步兵作战。美国人为此付出了很多，首先他们去除了斯图亚特坦克的M7型炮塔，替换为一个无顶盖的T87型炮塔，为的是增加即将放入的M2型75毫米火炮的俯仰角。这种新炮塔正面为38毫米，两侧和后方都为25毫米。弹药为反战车榴弹M66型和高爆榴弹M48，备弹药46发，辅助武器为一挺勃朗宁M1919A4型7.62毫米同轴机枪，它的动力装置为双卡迪拉克系列42型，16缸，汽油机，马力220。该车辆的最大速度可以达到每小时58千米，非常优秀。整体防护和M5斯图亚特坦克一样。

## 补充材料1
【M8自行火炮】除了自行火炮之外，M8还曾有一个轻型坦克歼击车项目M8A1。M8A1是美军为了填补坦克歼击车项目的空白而改装的应急之作，其主武器为一门谢尔曼坦克使用的75毫米炮。但随着M18坦克歼击车项目的神速推进，M8A1未能量产。【M8A1自行火炮】M8自行火炮从1942年9月进入量产，到1944年1月停产，总产量为1778辆，随美军征战于西线和太平洋战场。M8最大的价值体现在山地作战中，特别是在意大利战役期间。机动性能极为出色的M8可以在多山的亚平宁半岛肆意驰骋，作为一辆自行火炮经常在战斗中把中型坦克甩在身后。在阿尔卑斯山地及太平洋岛屿丘陵的地形中，M8经常可以爬上山坡近距离轰击德军和日军的碉堡火力点，给己方步兵带来最直接的支援。在二战结束后，M8自行火炮还参加了法越战争，并在阿尔及利亚军队中一直服役到60年代。

## 补充材料2
行驶性能和武器效能方面：最大速度甚至可以接近70KM 但是实际行驶起来也不会达到52KM。M8A1是由轻型坦克改装而来 速度也完全和轻型坦克相同。他可以用来帮助你翻越各种地形。不过作为一种临时抱佛脚的炮塔弊端很多。首先这种炮塔方向机较为简陋。并且无论是转速还是高低机性能都较为差劲。如果你想把它当正常坦克来用可就大错特错了。炮塔是用来微调的，并没有轻型坦克炮塔那样的转速，玩起来和坦克歼击车根本是差不多的。不过得益于操作性优良的底盘他可以依靠转动车体来瞄准目标，这和M10坦克歼击车的玩法特点非常相似。他的炮塔慢并不影响他瞄准自己的敌人 你可以通过挪动自己作为轻型坦克灵活的身躯来辅助你瞄准你的敌人 这会比你慢悠悠的转动炮塔要快得多。
