# 中文复杂检索文档

## 背景材料1
硫磺皂虽有去螨虫效果,却不能长期用,长期使会造成皮肤干燥、破坏皮脂平衡、破坏皮肤天然弱酸性保护层,不利于肌肤的健康,因此要注意硫磺皂的使用时间,不要天天用。使用完硫磺皂之后可能会觉得脸部比较干燥,因此使用完之后一定要涂上润肤霜。硫磺皂比较适合的肌肤是油性肌肤,因为能中和酸性肌肤,所以干性肌肤尽量不要选择这种方式去螨虫。再就是用硫磺皂洗澡洗手时不要佩戴银饰,因为硫磺皂一旦和银饰触碰,就容易发生化学反应的,会导致银饰颜色发红或发黑,虽然对金饰影响不明确,但是为了首饰的光亮度,还是尽量在使用硫磺皂时不要戴首饰或者手表了。

## 背景材料2
1、药皂中其实就是填加少量的苯酚,所以具有一定的消毒杀菌的作用。硫磺皂有杀细菌、真菌和螨虫的功效,对一些皮肤病有一定的预防和辅助治疗作用 2、药皂可杀灭金黄色葡萄球菌、大肠杆菌、绿脓杆菌、白色念株菌、溶血性链球菌、肝炎病菌等。硫磺皂能抑制皮脂分泌、杀灭细菌、霉菌、寄生虫等特殊功能,对皮肤病有辅助治疗作用 3、除菌成份能有效杀灭皮肤和衣物上常见的有害细菌,帮助预防由细菌引起的疾病、感染、过敏和汗臭,可以用来洗手,洗澡洗内衣裤。使用硫磺皂能洁身,兼具去屑止痒清痱。长期使用可防治皮肤瘙痒、疥疮、脚气等症。 4、上海药皂是碱性的,含有药物成分 如果你是油性的肤质就比较适宜。如果你皮肤属于干燥型的就少用。硫磺皂比临床上用的硫磺栓的浓度低多了,但也不建议长期连续使用。

## 补充材料1
病情分析: 硫磺皂有杀细菌、真菌和螨虫的功效,对一些皮肤病有一定的预防和辅助治疗作用。长期使用硫磺皂不会导致菌群失调,降低皮肤抵抗力,但它容易使皮肤缺水,引起皮肤干燥和脱皮等现象,故不建议长期使用。意见建议:硫磺皂比临床上用的硫磺栓的浓度低多了,但也不建议长期连续使用。建议间断使用,如开始时可以一天两次,症状减轻后可改为一天一次、三天一次等,逐渐间断直至停止。长期使用硫磺皂不会导致菌群失调,降低皮肤抵抗力,但它容易使皮肤缺水,引起皮肤干燥和脱皮等现象。 长期用硫磺皂洗对皮肤有一定的影响,如果你不是患有疥疮一类皮肤病的话,没有必要天天用它来洗,硫磺皂本身具有杀菌的作用,但是用长了会影响皮肤的光泽度。
